# Sign2Braille
Το project μας για τον Διαγωνισμό ΕΛΛΑΚ 2026.

Σύστημα ΑΙ το οποίο αναγνωρίζει την νοηματική και μετατρέπει τα σύμβολα της νοηματικής, με τη χρήση κατάλληλου μηχανισμών σερβοκινητήρων σε «ακίδες» γραφής Braille. Με τον τρόπο αυτόν, ένα άτομο με προβλήματα ακοής θα μπορεί να επικοινωνεί live με ένα άτομο που αντιμετωπίζει προβλήματα όρασης.

Η λογική του Project είναι απλή:

Δύο άτομα, το ένα προβλήματα ακοής και το άλλο με προβλήματα όρασης θα πρέπει να επικοινωνήσουν.

Το άτομο με προβλήματα ακοής βρίσκεται μπροστά σε μια κάμερα και επικοινωνεί στη νοηματική.

Η κάμερα συλλαμβάνει τις χειρονομίες και αναγνωρίζει τα γράμματα / χαρακτήρες.

Οι χαρακτήρες μεταδίδονται σε κατάλληλη συσκευή λήψης, η οποία τους αναγνωρίζει και τους μετατρέπει με τη βοήθεια κατάλληλης διάταξης με σερβοκινητήρες σε "ακίδες" γραφής Braille.

Το άτομο με προβλήματα όρασης ακουμπάει με το χέρι του τις "ακίδες" και αντιλαμβάνεται το μήνυμα.
