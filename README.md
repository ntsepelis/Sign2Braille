# Sign2Braille
Σύστημα ΑΙ το οποίο αναγνωρίζει την νοηματική και μετατρέπει τα σύμβολα της νοηματικής, με τη χρήση κατάλληλου μηχανισμών σερβοκινητήρων σε «ακίδες» γραφής Braille. Με τον τρόπο αυτόν, ένα άτομο με προβλήματα ακοής θα μπορεί να επικοινωνεί live με ένα άτομο που αντιμετωπίζει προβλήματα όρασης.

Το project μας για τον Διαγωνισμό ΕΛΛΑΚ 2026.
